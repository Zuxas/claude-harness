## Summary
Obsidian is a free, open-source markdown editor (v1.12.7) used within the Zuxas Harness to view and edit a knowledge base. While it renders markdown files into an interconnected knowledge graph using [[wikilinks]] and provides features like graph view and live preview, it is explicitly noted that the plain markdown files themselves, not the Obsidian application, constitute the source of truth for the knowledge base located at `E:\vscode ai project\harness\knowledge\`.

*   **Definition:** Obsidian is a free, open-source markdown editor.
*   **Developer:** Obsidian.md.
*   **Version:** v1.12.7.
*   **Functionality:** It renders markdown files as an interconnected knowledge graph.
*   **Installation:** Installed via `winget` on 2026-04-15.

### Zuxas Harness Integration
*   **Role:** Within the Zuxas Harness, Obsidian functions as the viewer/editor for the knowledge base.
*   **Knowledge Base Location:** `E:\vscode ai project\harness\knowledge\`.
*   **Source of Truth:** The plain markdown files are the source of truth. [INFERRED] If Obsidian were to disappear, the knowledge base would remain functional using any standard text editor.
*   **Key Features Used:**
    *   Graph view (for visualizing connections between blocks).
    *   Sidebar navigation.
    *   Live markdown preview.
    *   Direct file editing (which is detectable by Claude Code in subsequent sessions).
